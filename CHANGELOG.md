# Kronos股票预测系统 - 更新日志

## v2.1.0 (2024-12-02)

### 🚀 新功能
- **多数据源支持**: 新增Tencent和Baostock作为数据获取备用源
- **5分钟预测优化**: 训练数据从3天扩展到20天，特征窗口调整为60分钟
- **实时日志输出**: GUI界面支持实时显示分析进度和详细日志

### 🔧 技术改进
- **数据源容错**: 实现AkShare → yfinance → Tencent → Baostock的自动切换机制
- **动态特征窗口**: 根据时间框架智能调整机器学习特征窗口大小
  - 5分钟: 12个时间点 (60分钟历史)
  - 15分钟: 8个时间点 (120分钟历史)  
  - 日线: 10个时间点 (10天历史)
- **网络稳定性**: 增强网络连接的重试和错误处理机制

### 📈 算法优化
- **机器学习模型**: Random Forest在5分钟预测中使用更长的训练历史
- **特征工程**: 针对不同时间框架优化技术指标计算窗口
- **预测准确性**: 通过增加训练数据量提升短期预测效果

### 🛠️ 系统优化
- **代码重构**: 清理无效数据和临时文件
- **版本管理**: 统一版本号系统，添加详细的版本信息
- **文档完善**: 更新代码注释和功能说明

### 🔍 Bug修复
- 修复网络数据获取失败时的程序崩溃问题
- 修复GUI界面在长时间分析时的卡顿问题
- 修复5分钟数据处理中的时间戳错误

## 系统要求

- Python 3.8+
- Windows 10/11
- 网络连接（用于数据获取）

## 支持的数据源

1. **AkShare** (主要) - 提供全面的A股数据
2. **yfinance** (备用) - 国际化数据源
3. **Tencent** (备用) - 腾讯财经接口
4. **Baostock** (备用) - 证券宝数据平台

## 技术栈

- **GUI**: Tkinter + matplotlib
- **机器学习**: scikit-learn (Random Forest)
- **技术分析**: 自研技术指标计算
- **数据处理**: pandas + numpy
- **深度学习**: PyTorch (Kronos模型，可选)
