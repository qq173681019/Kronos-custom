# Kronos股票预测系统 - 使用说明

## 程序功能
这是一个带有图形界面的股票预测系统，能够直接在程序中显示预测图表。

## 文件说明
- `KronosGUI.exe` - 带GUI界面的主程序（推荐使用）
- `prediction_example_simple.exe` - 命令行版本程序
- `prediction_gui.py` - GUI程序源代码
- `prediction_example_simple.py` - 命令行程序源代码

## 主要特性
✅ **直接显示图表** - 图表直接在程序内显示，无需打开外部图片文件
✅ **交互式界面** - 可输入股票代码，选择图表类型
✅ **多种图表类型** - 支持日线图和15分钟图
✅ **数据导出** - 自动保存CSV数据和图表文件
✅ **独立运行** - 无需安装Python或其他依赖

## 使用方法

### GUI版本（推荐）
1. 双击运行 `KronosGUI.exe`
2. 在"股票代码"框中输入股票代码（如：600977）
3. 选择图表类型：
   - 日线图：显示每日股价走势
   - 15分钟图：显示更细粒度的价格变化
4. 设置预测参数：
   - 历史数据长度：用于预测的历史数据天数
   - 预测长度：要预测的未来天数
5. 点击"运行预测"按钮
6. 程序会在右侧直接显示图表，包括：
   - 上图：历史价格（蓝色）+ 预测价格（红色虚线）
   - 下图：历史成交量（蓝色）+ 预测成交量（红色）
7. 可以使用图表下方的工具栏进行缩放、平移等操作
8. 点击"保存图表"可以将当前图表保存为图片文件
9. 点击"打开结果文件夹"查看生成的CSV数据文件

### 命令行版本
```bash
prediction_example_simple.exe [股票代码] [图表类型] [历史天数] [预测天数]
```
例如：
```bash
prediction_example_simple.exe 600977 daily 30 7
```

## 输出文件
程序运行后会在`data`文件夹中生成以下文件：
- `{股票代码}_historical_{图表类型}.csv` - 历史数据
- `{股票代码}_prediction_{图表类型}.csv` - 预测数据  
- `{股票代码}_prediction_chart_{图表类型}.png` - 图表文件

## CSV文件格式
生成的CSV文件包含以下列：
- `timestamps` - 时间戳
- `open` - 开盘价
- `high` - 最高价
- `low` - 最低价
- `close` - 收盘价
- `volume` - 成交量

## 注意事项
1. 本程序使用模拟数据进行演示，实际使用时可替换为真实股票数据源
2. 程序会自动创建`data`文件夹存储结果文件
3. 首次运行可能需要几秒钟时间启动
4. 支持Windows 7及以上版本系统
