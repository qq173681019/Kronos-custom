## 设计特点

### 颜色方案
- 背景：紫蓝渐变 (#667eea → #764ba2)
- 卡片：白色圆角
- 上涨：绿色 (↑)
- 下跌：红色 (↓)

### 交互特性
- ✅ 触摸按钮有反馈动画
- ✅ 股票切换实时更新
- ✅ 大号字体易读
- ✅ 响应式布局适配所有设备
- ✅ 离线可用

### 技术实现
- 纯HTML + CSS + JavaScript
- 无需后端API
- 预计算数据，加载快速
- 可部署到任何静态托管

## 部署到Vercel

### 方法1：项目根目录部署
```bash
cd /path/to/Kronos-custom
vercel deploy
```

### 方法2：直接部署移动端
```bash
cd mobile_reports
vercel deploy
```

### 访问
部署后即可在手机浏览器访问，支持：
- iOS Safari
- Android Chrome
- 微信内置浏览器
- 所有现代浏览器
