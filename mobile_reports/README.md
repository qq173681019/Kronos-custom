# Kronos移动端预测界面

## 📱 演示说明

这是Kronos股票预测系统的移动端演示界面。

### 文件说明

- `index.html` - 移动端首页
- `demo.html` - 交互式演示页面，展示多个股票的预测数据

### 本地查看

直接用浏览器打开 `demo.html` 即可查看移动端界面。

或者启动本地服务器：
```bash
python -m http.server 8000
```
然后访问: http://localhost:8000/demo.html

### 部署到Vercel

1. 将此 `mobile_reports` 文件夹内容作为根目录
2. 在Vercel中导入并部署
3. 设置为静态站点即可

### 特性

✅ 完全响应式设计
✅ 触摸优化的交互
✅ 渐变色彩UI
✅ 多股票切换
✅ 实时价格显示
✅ 预测趋势可视化

### 数据来源

演示数据来自现有的CSV预测文件，展示了系统的实际预测能力。
